# Gym Landing Page

A responsive gym website landing page built with HTML and CSS.

## Features

- Responsive header with mobile menu
- Hero section with call-to-action
- Classes section with an adaptive image grid
- Membership plans with different card styles
- Font Awesome icons for visual details

## Files

- `index.html` - main page structure
- `style.css` - page styling and responsive layout

## How To Use

Open `index.html` in a browser to view the project.

## Notes

- Class images are loaded from the project folder
- Font Awesome is loaded from a CDN
